# Habit Tracker – INFR3120 Assignment 3
A full CRUD Habit Tracking web application built using **Node.js**, **Express.js**, **MongoDB Atlas**, **Mongoose**, and **EJS**.  
This project was created for INFR3120 Assignment 3 and includes secure database setup, shared EJS templates, Bootstrap design, and cloud deployment.

Live Demo: **https://habittracker-assignment03.onrender.com**

#Features

- Public page that displays all habits  
- Add new habits  
- Edit and update habit details  
- Delete habits with confirmation page  
- Shared header and footer using EJS partials  
- Bootstrap and custom UI design  
- MongoDB Atlas database with secure `.env` configuration  
- Fully deployed on Render  
- Clean folder structure (MVC-like)

#Technologies Used

- **Node.js**
- **Express.js**
- **MongoDB Atlas**
- **Mongoose**
- **EJS (Embedded JavaScript Templates)**
- **Bootstrap**
- **Render (Cloud Hosting)**
